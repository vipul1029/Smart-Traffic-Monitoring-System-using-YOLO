# Vehicle-Detection-and-In-Out-Counting-using-YOLO
Vehicle Detection and In/Out Counting using YOLO (Computer Vision)

## 📌 Project Overview
This project uses YOLOv8 and OpenCV to detect vehicles and count them based on line crossing direction.

The system tracks vehicles using unique IDs and counts:
- IN vehicles (moving downward)
- OUT vehicles (moving upward)

---

## 🚀 Features
- Real-time vehicle detection
- Vehicle tracking using YOLOv8
- Line crossing detection
- IN / OUT counting system
- Works on video input

---

## 🛠️ Technologies Used
- Python
- OpenCV
- YOLOv8 (Ultralytics)

---
## Output
- Detects vehicles in video
- Tracks each vehicle with unique ID
- Counts IN and OUT vehicles
- Displays results in real-time
